# Create Conda Enviroment With Miniconda
I am Creating a Miniconda Enviroment for DESeq2 data analysis with R and Bioconductor in my windows systems using Windows Subsystem Linux  (WSL)

## Code in WSL terminal 
- Start the WSL in Terminal
= Type the folllowing to create a conda enviroment for R4.5 name as R4.5WS
```{cmd}
conda create --name R4.5WS


```
-Activate the R4.5WS conda enviroment as follows
```{CMD}
conda activate R4.5WS
```
